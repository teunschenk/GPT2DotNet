# GPT2DotNet

Based on the GPT-2 architecture, GPT2DotNet is a C# implementation of the GPT-2 language model. It allows you to generate text based on a given prompt, making it useful for various natural language processing tasks.
This is for learning how to implement a transformer-based language model in C#. The project includes the necessary components to load a pre-trained GPT-2 model and generate text based on user input.
Inference only, no training code is included. The project is designed to be simple and easy to understand, making it a great resource for those interested in learning about transformer models and their implementation in C#.
